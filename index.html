Clothing Sales Forecasting with SARIMA and LSTM models

In sales, forecasting allows us to adjust stratagies and make decisions based on accurate estimates of the future. This project implements a SARIMA model and a neural network that uses LSTM cells (a state-of-the-art model based on deep learning) for time series forecasting of clothing sales.

Github repository

Credit Card Fraud Detection with LightGBM

Fraud detection is a classification problem that focuses on detecting anomalies in data. ML models do this by training themselves on many examples of fraudulent and non-fraudulent cases. In this project, I created a LightGBM model that detects frauds in credit card transactions with high precision and recall.

Github Repository

Sales analysis with SQL and Power BI

This project employed Power BI and SQL to obtain insights about the sales of a pizza store. SQL queries were written to calculate KPIs and other important metrics, such as total revenue, number of orders, and average order value. Power BI was then used to visualize these metrics through an interactive dashboard.

Github Repository

Dog breed Classification with TensorFlow

This project makes use of transfer learning to create a model that accurately classifies 120 different dog breeds from images. The model employed was MobileNet V2, which was pre-trained on the ImageNet dataset.
